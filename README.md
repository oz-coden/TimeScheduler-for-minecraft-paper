# TimeScheduler for Minecraft Paper

Minecraft Paper サーバー内で、現実世界でちょうどX時になったときにお知らせしてくれる時報機能と、現実時間やゲーム内時間に準じたリマインダーを実行できる機能を提供します。

## 機能
* 今何時？という疑問を解消できるように、キリのいい時間に時報を出します。（この機能はオンオフ可能）
* 現実時間（分・時間）やゲーム内時間（Tick・日）を指定してリマインダーを設定できます。
* プレイヤーのMinecraftクライアントの言語設定を検出し、日本語（`ja_jp`）や英語（`en_us`）で適切なメッセージを表示します。また、必要であれば翻訳ファイルを用意してその他の言語についても登録することができます。

## 導入方法
1. GitHubの [Releases](../../releases) ページから最新の `TimeScheduler-x-x-x.jar` をダウンロードします。
2. サーバーの `plugins` フォルダに配置します。
3. サーバーを起動（または再起動）すると、自動的にプラグインの名前のフォルダの中に `config.yml` と `lang/` フォルダが生成されます。

## コマンド
| コマンド                                                       | 説明                                                    |
|------------------------------------------------------------|-------------------------------------------------------|
| `/schedule get (<target> <schedule-type>)`                 | 設定済みのスケジュールを確認します。引数を追加することで、フィルターできます。               |
| `/schedule set <target> <schedule-type> <time> <messages>` | スケジュールを設定します。                                         |
| `/schedule remove <id>`                                    | IDを指定して、設定したスケジュールを削除します。（IDは`/schedule get`で特定できます）  |

## ⚙️ 設定 (Configuration)
`plugins/TimeScheduler/config.yml` で基本設定を変更できます。  
また、`lang/` フォルダ内の `.yml` ファイルを編集することで、すべてのメッセージを自由にカスタマイズ可能です。  
もし、言語を追加したい場合には、`lang`フォルダの中に対応したコードの名前の翻訳済みymlファイルを配置してください。
